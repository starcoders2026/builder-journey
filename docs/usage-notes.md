## How to Review the Repository

1. Start with the main README file
2. Check the project summary
3. Review the method notes
4. Look at the training and evaluation notes
5. Check the results folder for outputs
6. Review the project checklist for progress

## Suggested Reading Order

- docs/project-summary.md
- docs/method-notes.md
- docs/training-notes.md
- docs/evaluation-notes.md
- docs/project-checklist.md
