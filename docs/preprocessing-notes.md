# Preprocessing Notes

## Purpose

This file is used to record data preprocessing steps used before model training.

## Common Preprocessing Steps

- Image resizing
- Image normalization
- Class folder organization
- Data split preparation
- Removing unclear or duplicated images
- Checking image format consistency
- Preparing input data for the model

## Preprocessing Checklist

- [ ] Check folder structure
- [ ] Check class names
- [ ] Check image quality
- [ ] Check image sizes
- [ ] Remove duplicated or unclear files
- [ ] Prepare training and validation sets
- [ ] Confirm data can be loaded correctly

## Notes

Clear preprocessing notes make the training process easier to repeat and review later.
