# Clean Code Notes

## Purpose

This file is used to collect notes about writing code that is easier to read, understand, and maintain.

## What is clean code?

Clean code means writing code in a simple, organized, and readable way.

## Main Principles

- Use clear variable names
- Keep functions focused
- Avoid repeating the same logic
- Add comments only when needed
- Keep files organized
- Remove unused code
- Test small changes step by step

## Why it matters

Clean code makes projects easier to improve, debug, and share with other people.

## Current Focus

My current focus is learning how to write code that is not only working, but also understandable.
