# Learning Strategy Notes

## Purpose

This file is used to collect notes about learning technical topics in a more structured and practical way.

## Main Approach

The goal is not only to read or watch tutorials, but to turn each topic into a small practical output.

## Learning Method

1. Pick one topic
2. Learn the basic concept
3. Build a small example
4. Write short notes
5. Save the result in the repository
6. Review what can be improved

## Good Learning Habits

- Keep notes simple
- Practice with small examples
- Avoid jumping between too many topics
- Review older notes
- Connect learning with real project ideas
- Track progress regularly

## Current Focus

My current focus is learning by building small and clear examples instead of only collecting information.
