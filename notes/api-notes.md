# API Notes

## Purpose

This file is used to collect notes about APIs, data exchange, and application communication.

## What is an API?

An API allows two applications to communicate with each other.

For example, a frontend page can send a request to a backend server, and the backend can return data as a response.

## Main Topics

- HTTP requests
- GET and POST methods
- JSON data
- API endpoints
- Status codes
- Error messages
- API testing tools

## Basic Request Flow

1. The user performs an action
2. The frontend sends a request
3. The backend receives the request
4. The backend processes the data
5. The backend sends a response
6. The frontend updates the page
