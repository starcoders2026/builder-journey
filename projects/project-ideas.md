# Project Ideas

## Purpose

This file is used to collect small project ideas that can help improve my development skills and portfolio.

## Ideas

### 1. Personal Portfolio Website

A simple website that introduces my skills, projects, and learning progress.

### 2. Habit Tracker

A simple app for tracking daily habits and progress.

### 3. Task Manager

A basic task management app with add, edit, complete, and delete features.

### 4. Learning Dashboard

A small dashboard that shows learning goals, completed tasks, and weekly progress.

### 5. Notes Organizer

A simple note-taking structure for organizing technical notes and project documentation.

## Selection Criteria

- Useful for daily practice
- Simple enough to build step by step
- Good for portfolio presentation
- Easy to improve over time
